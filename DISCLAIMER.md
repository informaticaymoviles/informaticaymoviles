que el titular del dominio decida añadir enlaces de afiliación (ver punto 5). No es una
tienda: no vende, factura ni gestiona ningún producto. Únicamente agrega, compara y
enlaza información publicada por terceros.

automatizado semanal y pueden estar **desactualizados, ser incorrectos o haber

titular de este proyecto **no garantiza la exactitud** de ningún dato mostrado.
**Verifica siempre el precio, las existencias y las condiciones finales en la web
oficial de la tienda antes de comprar.**

de páginas web de terceros. La persona que despliegue o ejecute este proyecto es la

- Comprobar el `robots.txt` y los Términos de Servicio de cada web antes de scrapearla.

Los mantenedores de este proyecto no se hacen responsables del uso que terceros hagan
del código para acceder a webs sin respetar sus condiciones.
